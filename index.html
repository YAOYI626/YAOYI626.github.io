##出现新的问题

更新博客是出现新的问题，hexo部署出现警告
大概是这样

解决方案如下（主要是第二行命令）

rm -rf .git

git config –global core.autocrlf false

该警告一般在切换系统后由于不同系统的换行符不同所导致。
出现原因可能是主题在Linux下搭建所导致，更具体原因不明。
